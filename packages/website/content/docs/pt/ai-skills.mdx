O Vercel Doctor pode ser instalado como skill para assistentes de codificação com IA. Após a instalação, seu assistente de IA sugerirá automaticamente executar o Vercel Doctor após alterações em projetos Next.js.

## Instalação em uma linha

<DynamicCodeBlock lang="bash" code={SKILLS_COMMAND} />

Ou instale via curl:

```bash
curl -fsSL https://vercel-doctor.com/install-skill.sh | bash
```

## Ferramentas suportadas

O instalador detecta e instala automaticamente para todas as ferramentas suportadas no seu sistema:

| Tool        | Skill location                                 |
| ----------- | ---------------------------------------------- |
| Claude Code | `~/.claude/skills/vercel-doctor/`              |
| Amp Code    | `~/.config/amp/skills/vercel-doctor/`          |
| Cursor      | `~/.cursor/skills/vercel-doctor/`              |
| OpenCode    | `~/.config/opencode/skills/vercel-doctor/`     |
| Windsurf    | `~/.codeium/windsurf/memories/global_rules.md` |
| Antigravity | `~/.gemini/antigravity/skills/vercel-doctor/`  |
| Gemini CLI  | `~/.gemini/skills/vercel-doctor/`              |
| Codex       | `~/.codex/skills/vercel-doctor/`               |

O instalador também adiciona o skill ao diretório `.agents/` do projeto para contexto em nível de projeto.

## O que o skill faz

Quando ativado, o assistente de IA irá:

1. Conhecer o Vercel Doctor e o que ele escaneia
2. Sugerir executar `npx -y vercel-doctor@latest . --verbose --diff` após alterações
3. Focar primeiro em corrigir problemas que reduzem tempo de execução de funções e invocações
